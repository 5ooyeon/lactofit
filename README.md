## 기능

### NAV바

<hr>
: 각 항목으로 이동, 구글API 활용한 로그인, 팔로우/좋아요 시 오는 알림 기능 

### 메인페이지
<hr>

1. OPENAI API를 활용한 운동 검색

2. YOUTUBE API를 활용한 영상 검색, 댓글 달기

3. 오운완 커뮤니티 > 팔로우, 좋아요

### 마이페이지 
<hr>
: 오운완을 통한 스트릭, 스트릭 변경, 스트릭 변경 가격 뽑기, 프로필사진 수정, 회원 탈퇴 기능
